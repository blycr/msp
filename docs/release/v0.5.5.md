# MSP v0.5.5 Release Notes / 发布说明

## 核心更新

- 开发模式体验（Windows）
  - 新增 `scripts/dev.ps1`：前后端联动开发，后端 Go 文件变更自动重编译并重启
  - 开发模式默认关闭“自动打开浏览器预览”（不影响正式构建/发行包）
- 配置与黑名单
  - `config.json` 缺省值补齐：黑名单字段默认初始化，避免新旧配置不一致
  - 开发模式生成/修复 `bin/dev/config.json` 时自动补齐黑名单字段
- 前端与移动端联调
  - Vite dev server 支持局域网访问（便于手机端测试）
  - 列表分页条布局修复：Prev/Next 分居左右，页码居中
  - Footer 文案与结构重排：统一品牌与多语言文案，年份更新为 2026
- 文档与忽略规则
  - `.gitignore` 进一步完善：忽略本地运行产物、构建输出与常见本机配置目录，减少无用文件干扰

## 兼容性

- 无破坏性改动（兼容既有配置；缺省字段将自动补齐）

## English Summary

- Better dev workflow (Windows):
  - Added `scripts/dev.ps1` for combined frontend/backend development with Go auto-rebuild/restart
  - Dev mode disables auto-opening the preview page by default (release builds unaffected)
- Config & blacklist defaults:
  - Auto-fill missing blacklist fields to keep old/new configs compatible
  - Dev config bootstrap (`bin/dev/config.json`) auto-fixes missing fields
- Frontend/mobile iteration:
  - Vite dev server is reachable from LAN for phone testing
  - Pagination bar layout: Prev/Next on left/right, page indicator centered
  - Footer cleaned up (brand + i18n) and year updated to 2026
- Docs/ignores:
  - `.gitignore` refined to ignore local runtime artifacts and build outputs
