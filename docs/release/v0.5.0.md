# MSP v0.5.0 Release Notes / 发布说明

## 核心更新
- 后端重构
  - 将后端从单文件实现拆分为更清晰的内部模块（config / handler / media / server / util / web / types）
  - 改善可维护性，为后续功能演进打基础
- 前端工程化升级
  - 引入 Vite 构建，前端目录结构调整为 `web/src` 与 `web/public`
  - 增加 PWA 相关资源（manifest 与 Service Worker）
- 构建与分发
  - 增加 Windows 构建脚本（PowerShell），简化本地构建流程
  - 调整嵌入式静态资源组织方式，便于打包与发布

## 兼容性
- 对使用者无明显破坏性改动；如从源码构建，请以新的 `web/` 结构与脚本为准

## English Summary
- Backend refactor into clearer internal packages (config/handler/media/server/util/web/types)
- Frontend tooling upgrade: Vite build; reorganized to `web/src` + `web/public`
- Added PWA assets (manifest + service worker)
- Added Windows build script and adjusted embedded static assets layout
