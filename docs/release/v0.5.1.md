# MSP v0.5.1 Release Notes

## 中文

- 性能优化
  - 左侧文件列表与播放列表支持分页（默认每页 10 项），避免一次性渲染过多造成卡顿
  - 音频播放器与图片预览采用与视频一致的淡入过渡（仅 opacity，时长 0.25s），显著改善切换卡顿
- 体验优化
  - 明暗主题切换过渡统一为轻量淡入，尊重系统“减少动态效果”设置
  - 文档更新：补充 PWA 使用方法、从源码构建步骤、隐私提交流程
- 修复
  - 解决在大量媒体项下切换预览出现的动画不一致与顿挫问题

## English

- Performance
  - Pagination for both sidebar file list and playlist (10 items per page by default) to reduce DOM rendering pressure
  - Audio player and image preview now use the same lightweight fade-in transition as video (opacity-only, 0.25s) for smoother switching
- UX
  - Unified light/dark theme transition with accessibility-friendly motion handling
  - Docs updated: PWA usage, build from source steps, privacy-aware remote submission workflow
- Fixes
  - Addressed inconsistent and janky transitions when switching previews with large media sets
