# MSP v0.5.1

> 重点：大列表更流畅 + 过渡更丝滑 + 文档更完善  
> Highlight: smoother large libraries + consistent transitions + improved docs

---

## 中文（Release Notes）

### 亮点
- **性能优化**
  - 左侧文件列表与播放列表支持分页（默认每页 10 项），减少一次性渲染带来的卡顿
  - 音频播放器与图片预览采用与视频一致的轻量淡入过渡（opacity-only，0.25s），切换更顺滑
- **体验优化**
  - 明暗主题切换过渡统一为轻量淡入，并尊重系统“减少动态效果”设置
  - 文档补充：PWA 使用方法、从源码构建步骤、隐私提交流程
- **修复**
  - 修复大量媒体项下切换预览时动画不一致、顿挫的问题

### 兼容性
- 无破坏性改动（兼容既有配置与使用方式）

---

## English (Release Notes)

### Highlights
- **Performance**
  - Pagination for sidebar list and playlist (10 items/page by default) to reduce DOM rendering pressure
  - Audio player and image preview now use the same lightweight fade-in as video (opacity-only, 0.25s)
- **UX**
  - Unified theme transition with accessibility-friendly motion handling (“reduce motion” aware)
  - Docs: PWA usage, build-from-source steps, privacy-aware remote submission workflow
- **Fixes**
  - Addressed inconsistent/janky transitions when switching previews with large libraries

### Compatibility
- No breaking changes
