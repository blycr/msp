# MSP v0.4.0 Release Notes / 发布说明

## 核心更新
- 多语言（I18N）
  - 前端新增中英双语界面（EN / ZH），常用按钮与提示支持语言切换
- 搜索与排序体验
  - 文件名搜索增强：拼音 / 正则 / 模糊匹配
  - 新增排序选项，并修复相关交互与逻辑问题
- UI 与播放体验
  - 优化整体布局与动画表现，修复自动播放等体验问题
  - 统一播放器容器与圆角风格，细节视觉更一致
- 文档
  - 增加英文 README，并对文档结构与入口做了调整

## 兼容性
- 无破坏性改动

## English Summary
- Added EN/ZH UI translations (I18N)
- Improved search (pinyin/regex/fuzzy) and added/adjusted sorting options
- UI/UX polish: layout, animations, autoplay fixes, and consistent styling
- Documentation updates (English README and entry adjustments)
