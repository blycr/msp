## MSP v0.2.0

这一版主要在“播放体验”和“发布/维护友好度”上做了打磨～

### 变更内容

- PC 端视频全屏体验改进：修复 Plyr 全屏出现大黑边问题
- 流媒体接口更稳：`/api/stream` 对常见媒体扩展名返回更明确的 `Content-Type`
- 前端资源结构整理：静态资源统一放在 `web/static` 并通过 embed 内置
- 隐私更安全：`config.json` 作为运行时本机配置不再提交；提供 `config.example.json` 供参考
- UI 更一致：除视频预览外，隐藏“填充模式”按钮

### 兼容性提示

- 若某些视频仍无法播放，通常是编码不被浏览器支持（例如 HEVC/H.265、AV1、AC-3/E-AC-3 等）；可在页面的编码探测提示中查看容器与音视频编码信息。
